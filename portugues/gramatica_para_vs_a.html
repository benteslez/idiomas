Para vs A — Distinção Crítica

Sintaxe · Armadilha
Gramática portuguesa (PB) · CEFR B1–B2
Destino · finalidade
A A
La distinción para vs a es una de las trampas más sistemáticas del portugués para hispanohablantes. El español usa "a" para destino y "para" para finalidad; el portugués reorganiza esta distribución de forma distinta, especialmente en PB, donde para domina como preposición de destino.
🚨
Regla de oro para hispanohablantes: donde el español usa "a" para destino, el PB generalmente usa para. "Voy a casa" → "Vou para casa". "Voy al banco" → "Vou ao banco" (con artículo, sigue "a") o "Vou no banco" (PB hablado muy informal).

Distinción tradicional: en PE clásico, a es destino breve (volveré pronto) y para es destino con permanencia ("Vou a Paris para estudar"). En PB hablado, esta distinción se debilita: para domina ampliamente.
Para indicar destino físico, el PB usa tres opciones según contexto y registro: para (dominante), a (formal o estancia breve), y em/no/na (PB hablado muy informal con verbos como ir).
Las tres opciones de destino

Para (PB dominante)

Destino con permanencia significativa. La forma más natural y frecuente en habla.

Vou para casa.

Vou para o trabalho.

Vou para São Paulo.

Vou para a Bahia nas férias.

A (formal / breve)

Destino con estancia breve (sigue PE). Más formal en PB.

Vou ao mercado.

Vou à praia.

Fui a Paris e voltei no dia seguinte.

PB oral: em + lugar
Vou no banco. (PB hablado, muy informal)
Voy al banco.
Vou na padaria.
Voy a la panadería.
Vou no shopping.
Voy al shopping.
⚠️
Em + lugar con ir es PB popular: "vou no banco" es agramatical en norma escrita (debería ser ao banco o para o banco), pero es muy común en PB hablado. NO usar en escritura culta.

PB hablado: la distinción se debilita. En PB moderno, para cubre ambas funciones: estancia larga o breve. Si quieres parecer más formal o respetar la norma, usa la distinción tradicional. En habla cotidiana: para es lo natural.
En la función de finalidad (= "para qué", objetivo), el portugués usa exclusivamente para. Esto coincide con el español "para". A nunca aparece con valor de finalidad.
Para + infinitivo / sustantivo (finalidad)
Estudo para aprender.
Estudio para aprender.
Vim para ajudar.
Vine para ayudar.
Vou para trabalhar.
Voy para trabajar / a trabajar.
Esse remédio é para a dor de cabeça.
Este medicamento es para el dolor de cabeza.
É para seu próprio bem.
Es para tu propio bien.
Para que + subjuntivo (finalidad subordinada)
Falo devagar para que você entenda.
Hablo despacio para que entiendas.
Trouxe o livro para que ele leia.
Traje el libro para que lo lea.
Para + inf. impessoal vs para + inf. pessoal
É difícil para entender. (mismo sujeto o impersonal)
Es difícil para entender.
É difícil para vocês entenderem. (infinitivo pessoal)
Es difícil que ustedes entiendan.
⚠️
NUNCA "a + finalidade": el español dice "esto sirve a estudiar" (raro) o "esto sirve para estudiar". En portugués SIEMPRE para: serve para estudar, no serve a estudar.
Para destinatarios y personas, el portugués distingue: para indica destinatario o beneficiario; a indica complemento indirecto (norma) o destinatario en registro escrito formal. En PB hablado, para domina ampliamente.
Destinatario / beneficiario: para
Esse presente é para você.
Ese regalo es para ti.
Comprei flores para minha mãe.
Compré flores para mi madre.
Fiz isso para ela.
Lo hice para ella.

Verbos de transferencia: dar / dizer / enviar
Dei o livro para a Maria. / Dei à Maria. (formal)

💡
PB hablado: el complemento indirecto se construye casi siempre con para en habla: falei para ela, escrevi para o João, mostrei para vocês. La forma con a/ao/à queda en escritura formal o discurso cuidado.
Distinción matiz: ir a vs ir para alguém
Vou a ele. (raro: dirigirse a alguien — formal)
Voy a él. (formal/raro)
Vou para ele. (= me dirijo hacia él, fin de movimiento)
Voy hacia él.
Vou para o lado dele. (= a su lado)
Voy a su lado.
En la función temporal, a aparece con horas exactas y franjas horarias (às 8 horas, das 9 às 18h). Para aparece con plazos límite (para amanhã, para sexta-feira).
A con horas (norma universal)
Às oito horas começa a reunião.
A las ocho horas empieza la reunión.
Trabalho das 9 às 18 horas.
Trabajo de las 9 a las 18 horas.
À uma hora da tarde.
A la una de la tarde.
Ao meio-dia / à meia-noite.
A mediodía / a medianoche.
Para con plazos límite (deadline)
O trabalho é para amanhã.
El trabajo es para mañana.
A reunião está marcada para sexta-feira.
La reunión está marcada para el viernes.
Tenho que terminar para a próxima semana.
Tengo que terminar para la próxima semana.
Estamos planejando uma viagem para dezembro.
Estamos planeando un viaje para diciembre.
Para indicar período futuro próximo
Vou ficar aqui por uma semana. (duración con por)
Voy a quedarme aquí por una semana.
Adiei a viagem para a próxima semana. (plazo)
Pospuse el viaje para la próxima semana.
⚠️
Resumen temporal: a / às = punto exacto en el tiempo (horas). Para = plazo, fecha límite. Por = duración. Desde = inicio. Em = ubicación temporal (en agosto, en 2020).

En PB hablado, para se reduce frecuentemente a pra (o incluso pra + artículo: pro, pra, pros, pras). Es muy productivo en habla y aparece sistemáticamente en literatura coloquial, ficción dialogada, redes sociales.
Pra: reducción oral de para
Norma escritaPB hablado
É para você.É pra você.
Vou para a casa.Vou pra casa.
Para o Pedro.Pro Pedro.
Para a Maria.Pra Maria.
Para os meninos.Pros meninos.
⚠️
Pra en escritura: en escritura formal (redação, prensa, ensayo), usa siempre para. En mensajes, ficción dialogada, redes sociales, pra es aceptable. No es una falta de ortografía: es una reducción reconocida del habla.
