Formação de Palavras

Léxico
Gramática portuguesa (PB) · CEFR B2–C1
Sufijos · prefijos · derivación
AA
La formação de palavras en portugués se realiza principalmente por derivación (sufijos y prefijos) y composición (unión de dos palabras). Conocer los sufijos y prefijos productivos permite inferir significados de palabras nuevas y crear neologismos espontáneos. Es competencia léxica clave para B2–C1.

💡
Productividad: los sufijos más productivos en PB moderno son -ção, -mento, -dade, -ismo, -izar, -ar, -inho. Permiten formar neologismos espontáneos: privacização, empoderar, descomplicar, googlear (col.).
Los sufijos nominais forman sustantivos a partir de verbos, adjetivos u otros sustantivos. Cada sufijo aporta un matiz semántico (acción, proceso, cualidad, profesión, doctrina).
-ção (-são)
acción / efecto (de verbo)
educar → educação, ver → visão, ver → televisão.
-mento
acción / efecto
crescer → crescimento, pensar → pensamento.
-dade (-idade)
cualidad abstracta (de adj.)
livre → liberdade, igual → igualdade.
-eza
cualidad (de adj.)
belo → beleza, fraco → fraqueza.
-ura
cualidad / estado
doce → doçura, alto → altura, frio → frescura.
-agem
acción / colectivo
paisar → paisagem, ferrar → ferragem.
-ismo
doctrina / sistema
social → socialismo, modern → modernismo.
-ista
persona / adepto
jornal → jornalista, dent → dentista.
-ário / -eiro
profesión / lugar
arte → artista, livro → livraria, livreiro.
-ador / -edor
agente (persona o instr.)
trabalhar → trabalhador, ven → vendedor.
-ância / -ência
cualidad / acción
tolerar → tolerância, viver → vivência.
-ório / -ária
lugar / función
conservar → conservatório, secre → secretaria.
-aria / -eria
tienda / lugar de
livro → livraria, pão → padaria, pizza → pizzaria.
-zinho / -zinha
diminutivo (afectivo)
café → cafezinho, mão → mãozinha.
-ão / -ona
aumentativo
carro → carrão, mulher → mulherona.
Los sufijos adjetivais forman adjetivos a partir de sustantivos, verbos u otros adjetivos. Aportan matices como posibilidad, posesión, relación, abundancia.
-vel / -ável / -ível
posibilidad / capacidad
amar → amável, fazer → factível.
-oso / -osa
abundancia / posesión
amor → amoroso, gosto → gostoso, ódio → odioso.
-al
relativo a
natur → natural, glob → global, ment → mental.
-ivo / -iva
tendencia / función
produzir → produtivo, ativ → ativo, criar → criativo.
-ento
cualidad (de sust.)
calor → calorento, pó → poento, friorento.
-eiro / -eira
relativo a / oficio
brasileiro, fofoqueiro, festeiro.
-ano / -ana
origen / pertenencia
italiano, paraense, mexicano.
-ês / -esa
origen
português, francês, japonês.
-ino / -ina
relativo a (raro)
argentino, paulistano, marinho.
-udo / -uda
posesión enf. (col.)
barbudo, peludo, sortudo.
-ado / -ido
participio adj.
cansado, perdido, abandonado.
-ante / -ente
cualidad activa
brilhante, importante, urgente.
-ico / -ica
relativo a (técnico)
romântico, prático, geográfico.
-iano
relativo a (autor)
freudiano, machadiano, sebastianista.
Ejemplos en uso
Um problema solucionável = que se puede solucionar.
Uma pessoa amorosa = llena de amor.
A vida moderna = relativa a la modernidad.
Los sufijos verbais forman verbos a partir de sustantivos o adjetivos. Permiten "verbalizar" cualquier concepto. Son altamente productivos (especialmente -izar).
-izar
hacer / transformar en
moderno → modernizar, real → realizar.
-ificar
hacer (formal)
claro → clarificar, simples → simplificar.
-ejar
acción repetida / lúdica
gota → gotejar, bra → bracejar.
-ecer
volverse / aumentar
escur → escurecer, vermelho → avermelhecer.
-itar / -ear
acción intensiva / suave
debilitar, transit → transitar, golpe → golpear.
en- ... -ar
causativo
grand → engrandecer, gord → engordar.
a- ... -ar
aproximación
tarde → atrasar, próx → aproximar.
Productividad de -izar
global → globalizar, computador → computadorizar, profissional → profissionalizar.
Neologismos: facebookizar, instagramizar, googlear (algunos resultados con -ar).
-izar es muy productivo: el PB acepta neologismos con -izar para cualquier nuevo concepto: privatizar, terceirizar, viralizar, monetizar, descentralizar, digitalizar. En tecnología, política, marketing.
Los prefijos añaden un matiz semántico al radical sin cambiar la clase de palabra. Son altamente productivos en formación de neologismos.
in- / im- / i-
negación
feliz → infeliz, possível → impossível, legal → ilegal.
des-
acción inversa
fazer → desfazer, conectar → desconectar.
re-
repetición
fazer → refazer, ler → reler, ver → rever.
super- / sobre-
por encima / superlativo
super-homem, sobrenatural, supermercado.
sub-
debajo / inferior
submarino, subterrâneo, subgerente.
pré- / pre-
antes
pré-história, prever, precolombino.
pós- / pos-
después
pós-graduação, pós-guerra, posdoctoral.
anti-
contra
anti-herói, antibiótico, antissocial.
contra-
en oposición
contraproducente, contrapeso, contrariar.
hiper- / mega-
superlativo (col. PB)
hipermercado, megaprojeto, megabacana.
auto-
por sí mismo
autoestima, autocrítica, autoescola.
inter- / entre-
entre
internacional, intercâmbio, entrevista.
multi- / pluri-
muchos
multifuncional, pluricultural, multinacional.
ex-
ex-, antiguo
ex-presidente, ex-marido, ex-aluno.
extra-
fuera / superlativo
extraordinário, extraescolar, extrasensorial.
vice-
en lugar de
vice-presidente, vice-líder.
⚠️
Guion en prefijos: según el Acordo Ortográfico de 1990, algunos prefijos usan guion (pré-, pós-, vice-, ex-, super- ante vocal) y otros no (anti-, auto-, contra-, infra-, inter- en general unidos). Reglas precisas en cada caso.
Otros procesos: composição (unión de palabras), aglutinação (fusión total), siglas / acrónimos, conversión (cambio de clase sin sufijo).
Composição (con guion o sin)
Sust. + sust.
unidad compleja
peixe-espada, palavra-chave, pé-de-meia.
Verbo + sust.
agente / instrumento
guarda-chuva, beija-flor, quebra-cabeça.
Adj. + sust.
valoración
má-fé, boa-vontade, livre-arbítrio.
Sust. + adj.
color compuesto
azul-claro, vermelho-vivo, cor-de-rosa.
Numeral + sust.
cantidad
terça-feira, segunda-feira, sete-estrelo.
Aglutinação (fusión total)
em + boa hora → embora
para + a → pra (en habla)
por + isso → por isso (se mantiene separado)
Siglas y acrónimos
PT (Partido dos Trabalhadores), USP (Universidade de São Paulo), PUC, CPF.
Genéricos: UTI, RH, BO, RG, CEP, CNPJ.
Conversión (cambio de clase sin sufijo)
Verbo → sust.: o jantar, o poder, o querer.
Adj. → sust.: o belo, o útil, o justo.
Sust. → adj.: cores laranja, rosa, vinho (color).
