🔀 Conditionals · Condicionales

Linguaskill B2 · B2–C1

Las cuatro estructuras condicionales del inglés — real (verdad general y futuro probable), irreal (hipótesis presente y pasada) y mixto (causa pasada · consecuencia presente) — con conectores alternativos.

A A 100%

Condicionales reales · Real Conditionals

Cubren situaciones que son ciertas, posibles o probables. El Tipo 0 expresa verdades generales y leyes científicas; el Tipo 1 habla de un futuro real y específico. En ambos, la if-clause va en presente y la principal en presente o con will / can / may / imperativo.

🎯Linguaskill: núcleo de los huecos de Reading · Use of English y de las advertencias/promesas en Writing P1 (email). Si fallas la concordancia if + present / will, perderás los puntos más fáciles del examen.
🔧
A. Forma · Form
Estructura canónica de los condicionales reales
Tipo If-clause Main clause Ejemplo Glosa ES
Tipo 0
verdad general		 If + present simple present simple If you heat ice, it melts. Si calientas el hielo, se derrite.
Tipo 1
futuro real		 If + present simple will + infinitive If it rains, we 'll cancel the picnic. Si llueve, cancelaremos el pícnic.
Tipo 1 (modal)
matiz		 If + present simple can / may / might / should + inf. If you finish early, you can leave. Si terminas pronto, puedes irte.
Tipo 1 (imperativo)
instrucción		 If + present simple imperative If you see John, tell him to call me. Si ves a John, dile que me llame.
Variante
orden inverso		 main clause + if-clause (sin coma) We 'll cancel the picnic if it rains. Cancelaremos el pícnic si llueve.
⭐
B. Usos · Uses
Cuándo elegir Tipo 0 o Tipo 1
Tipo 0 · Verdades generalesleyes universales
  • Ciencia: If water reaches 100 °C, it boils.
  • Hábitos: If I drink coffee at night, I can't sleep.
  • Instrucciones: If the alarm rings, leave the building.
Tipo 1 · Futuro realpredicción concreta
  • Predicción: If you train every day, you'll improve fast.
  • Plan condicional: If the meeting ends early, we'll go for lunch.
  • Advertencia: If you don't book now, you'll lose the seat.
  • Promesa: If you help me, I'll cook dinner.
Modales en la principalmatiz / probabilidad
  • may / might: incertidumbre — If she calls, I may pick up.
  • should: consejo — If you feel sick, you should rest.
  • can: permiso/capacidad — If you're 18, you can vote.
  • must: obligación — If you're driving, you must wear a seatbelt.
Imperativo en la principalinstrucciones · órdenes
  • If you have any questions, ask me.
  • If it's raining, take an umbrella.
  • If in doubt, don't click the link.

D. Trampas para hispanohablantes
Errores típicos por interferencia del español
!
Nunca "will" en la if-clause

El error más penalizado del Tipo 1: *If it will rain, we'll stay home → If it rains, we'll stay home. Tras if el futuro se expresa con presente simple, como en español con si.

!
"If" vs "when": probable vs seguro

Usa when si el evento es seguro (When I arrive, I'll call you — voy a llegar), y if si solo es posible (If I arrive on time, I'll call you — quizá no llegue). En español ambos se traducen como cuando / si.

!
Coma sí · coma no

Cuando la if-clause va primero, lleva coma: If it rains, we'll stay. Si va al final, no lleva coma: We'll stay if it rains. Los hispanohablantes tienden a omitir la coma inicial.

!
Confundir Tipo 0 con Tipo 1

Tipo 0 = ley universal (If you heat ice, it melts). Tipo 1 = un evento concreto futuro (If you heat the soup, it'll be ready). No traduzcas el español "se derrite / se derretirá" mecánicamente.

!
"Si no" no siempre es "if … not"

El español si no equivale a if … not o, más elegantemente, a unless: Si no te apuras, perderás el tren = If you don't hurry, you'll miss the train = Unless you hurry, you'll miss the train.

Condicionales irreales · Unreal Conditionals

Hablan de situaciones contrafactuales o improbables: lo que pasaría si las cosas fueran distintas. El Tipo 2 (hipótesis presente o futura irreal) y el Tipo 3 (hipótesis pasada contrafactual) usan el past simple y el past perfect con valor de subjuntivo — no son tiempos pasados reales.

🎯Linguaskill: imprescindibles para el matiz argumental en Writing P2 (essay) — If governments invested more, … — y para responder a hipótesis en Speaking P4 — What would you do if you were in charge?
🔧
A. Forma · Form
Estructura canónica de los condicionales irreales
Tipo If-clause Main clause Ejemplo Glosa ES
Tipo 2
presente irreal		 If + past simple would + infinitive If I had more time, I would travel more. Si tuviera más tiempo, viajaría más.
Tipo 2 (were)
subjuntivo formal		 If + were (todas las personas) would + infinitive If I were you, I would apologise. Si yo fuera tú, me disculparía.
Tipo 2 (modal)
matiz		 If + past simple could / might + inf. If we left now, we could catch the 5 o'clock train. Si saliéramos ahora, podríamos coger el tren de las 5.
Tipo 3
pasado irreal		 If + past perfect would have + past participle If I had studied harder, I would have passed. Si hubiera estudiado más, habría aprobado.
Tipo 3 (modal)
matiz		 If + past perfect could / might have + p.p. If you had told me, I could have helped. Si me lo hubieras dicho, habría podido ayudar.
⭐
B. Usos · Uses
Qué expresa cada tipo irreal
Tipo 2 · Presente irrealhipótesis improbable
  • Hipótesis improbable: If I won the lottery, I'd buy a house by the sea.
  • Situación contraria a la realidad: If I were taller, basketball would be easier.
  • Consejo: If I were you, I'd see a doctor.
  • Deseo disfrazado: If only I had a free weekend.
Tipo 3 · Pasado irrealcontrafactual
  • Arrepentimiento: If I had saved more, I wouldn't be in debt.
  • Reproche: If you had listened, we wouldn't be lost.
  • Análisis de lo no ocurrido: If the goalkeeper had stayed central, they'd have won.
  • Especulación histórica: If Napoleon had won at Waterloo, Europe would have changed.
Modales (could · might)posibilidad
  • could: habilidad/posibilidad — If she trained, she could win gold.
  • might: incertidumbre — If they invested in R&D, profits might rise.
  • Tipo 3 con could/might have: resultado posible no ocurrido — If we'd arrived on time, we might have caught the show.
Formal vs neutroregistro
  • Formal (essay): Were governments to act sooner, … (inversión)
  • Formal (essay): Had we acted then, the outcome would have differed.
  • Neutro: If governments acted sooner, …

D. Trampas para hispanohablantes
Errores típicos por interferencia del español
!
"If I were" (no "If I was") en el examen

If I were you es el subjuntivo correcto y el que el examinador espera en Writing. If I was you existe en habla muy informal británica, pero baja registro y se considera incorrecto en escritura académica. Regla: en condicionales irreales, usa were con todas las personas (I/he/she/it/we/they).

!
"Would" jamás en la if-clause

*If I would have time, I would travel → If I had time, I would travel. (Excepción única: peticiones corteses tipo If you would be so kind as to…, que no son condicionales sino fórmulas de cortesía.)

!
El "pasado" del Tipo 2/3 no es pasado real

If I had a car, I would drive = ahora no tengo coche (presente irreal). No traduzcas mecánicamente "si tuve un coche": el inglés usa el past simple como subjuntivo, igual que el español usa "si tuviera".

!
Tipo 3: pluscuamperfecto obligatorio

Para pasado contrafactual no basta con If I knew: hay que usar If I had known, I would have called. El español "si lo hubiera sabido" exige past perfect en inglés, no past simple.

!
"Would" no equivale al condicional español puro

El español "viajaría" es ambiguo: presente irreal (If I had time, I'd travel) o pasado pospuesto (He said he would travel). Identifica primero la cláusula if: si hay una, es condicional; si no, es reported speech.

!
Contracciones: 'd ambiguo

I'd puede ser I had o I would. En el Tipo 3 conviven los dos: If I'd known, I'd have called = If I had known, I would have called. Cuidado al desambiguar en lectura y dictado.

Condicional mixto y conectores · Mixed Conditional & Connectors

El condicional mixto combina los tiempos del Tipo 3 y del Tipo 2 para expresar una causa pasada con consecuencia presente. Además, varios conectores (unless, provided that, as long as, suppose…) permiten matizar la condición sin recurrir a if.

🎯Linguaskill: el condicional mixto y los conectores alternativos son marcadores de C1 — aparecen en huecos avanzados de Reading y suben la nota en Writing P2 al permitir argumentación matizada.
🔧
A. Forma · Form
Estructura del condicional mixto (3 → 2 y 2 → 3)
Tipo If-clause Main clause Ejemplo Glosa ES
Mixto 3 → 2
causa pasada · consecuencia presente		 If + past perfect would + infinitive If I had studied medicine, I would be a doctor now. Si hubiera estudiado medicina, ahora sería médico.
Mixto 3 → 2
marcador presente		 If + past perfect would + infinitive (+ now / today / still) If we had taken that flight, we 'd be in Tokyo by now. Si hubiéramos cogido ese vuelo, ya estaríamos en Tokio.
Mixto 2 → 3
estado presente · consecuencia pasada		 If + past simple / were would have + past participle If she were more cautious, she wouldn't have crashed. Si fuera más prudente, no habría chocado.
Mixto 2 → 3
característica permanente		 If + past simple would have + p.p. If he spoke English, he would have got the job last week. Si hablara inglés, habría conseguido el trabajo la semana pasada.
⭐
B. Usos · Uses
Cuándo elegir el mixto en lugar del Tipo 2 o 3 puros
3 → 2 · Lo que cambiaría hoycausa antigua · efecto actual
  • Arrepentimientos con efecto presente: If I hadn't quit my job, I'd be richer now.
  • Análisis biográfico: If she had accepted the offer, she'd be CEO today.
  • Marcadores típicos: now, today, still, by now, at the moment.
2 → 3 · Carácter que explica el pasadorasgo presente · resultado pasado
  • Carácter / rasgo permanente: If he weren't so impulsive, he wouldn't have lost his job.
  • Habilidad/idioma: If I spoke French, I would have understood the speech.
  • Hábitos generales: If she didn't smoke, she wouldn't have got ill.
Inversión formalregistro C1 · essay
  • Had + S + p.p. = If…had…: Had we invested then, we'd be ahead now.
  • Were + S + to-inf. = If…were to…: Were she to apply, she'd stand a real chance.
  • Estas inversiones omiten el if y suben el registro.
Cuándo NO usar el mixtocoherencia temporal
  • Si causa y consecuencia ocurren en el mismo plano temporal, usa Tipo 2 o Tipo 3 puro.
  • Si no hay marcador presente o pasado claro, el mixto puede sonar forzado.
  • El mixto exige siempre un contraste temporal explícito.
💬
C. Ejemplos reales · Real-life examples
Frases naturales con el mixto en contexto
If I had taken that job in London, I would be earning more now. Si hubiera aceptado ese trabajo en Londres, ahora ganaría más.
She would still be alive if the doctors had acted faster. Aún estaría viva si los médicos hubieran actuado más rápido.
If he weren't so impulsive, he wouldn't have lost his job. Si no fuera tan impulsivo, no habría perdido el trabajo.
Had we invested in renewables earlier, the country would be self-sufficient by now. [formal · essay] De haber invertido antes en renovables, el país ya sería autosuficiente.
[coloquial] We wouldn't be lost if you'd brought the map. No estaríamos perdidos si hubieras traído el mapa.

D. Trampas para hispanohablantes
Errores típicos con mixto y conectores
!
"Unless" = "if not": no añadas otra negación

*Unless you don't come, I won't go → Unless you come, I won't go (= If you don't come, I won't go). El español "a menos que no…" sí admite doble negación; en inglés es agramatical.

!
"Provided that" / "As long as": presente, no condicional

*Provided you would arrive on time → Provided you arrive on time. Funcionan como sustitutos de if en condicionales reales: if-clause en presente, principal con will / can / imperativo.

!
Marcadores temporales obligatorios en el mixto

Sin marcador (now, today, still, by now, anymore) la frase puede sonar a Tipo 3 mal hecho. If I had studied medicine, I would be a doctor mejora con now: If I had studied medicine, I would be a doctor now.

!
"Suppose / supposing" + past simple

Igual que un Tipo 2: Suppose you won the lottery, what would you do? No uses presente (*Suppose you win the lottery, what would you do?) salvo que sea claramente futuro real (entonces usa if, no suppose).

!
"But for" lleva sustantivo, no oración

But for your help, I would have failed (= If it hadn't been for your help…). El error frecuente es traducir "de no ser por que me ayudaste" como *But for you helped me. But for exige nombre o gerundio.

!
"Otherwise" no es un conector condicional puro

Otherwise = si no, de lo contrario, pero conecta dos oraciones independientes: Hurry up; otherwise we'll be late (no *Hurry up otherwise we are late). Lleva idealmente punto y coma o punto antes.

📐
E. Conectores alternativos · Beyond "if"
Unless, provided that, as long as, suppose, but for, otherwise
unless= if not
  • Excepción única: I won't go unless you come.
  • = If you don't come, I won't go.
  • Nunca con doble negación.
provided / providing (that)si y solo si
  • Condición estricta: You can borrow the car provided (that) you return it tonight.
  • Más formal que as long as; típico en contratos.
as long as / so long assiempre que · mientras
  • Condición flexible: I'll help you as long as you're honest.
  • Tono neutro; vale para escrito y hablado.
on (the) condition thatcon la condición de que
  • Muy formal: I'll lend you the money on condition that you pay it back next month.
  • Registro legal/administrativo; útil en Writing P2.
suppose / supposingsupón que · imagina que
  • Hipótesis: Suppose you won the lottery, what would you do?
  • Equivale a un Tipo 2: past simple tras suppose.
but for + Nde no ser por
  • Formal: But for your help, I would have failed.
  • = If it hadn't been for your help, …
  • Siempre con sustantivo o gerundio, nunca oración.
otherwisesi no · de lo contrario
  • Conexión entre oraciones: Hurry up; otherwise we'll be late.
  • Lleva idealmente punto y coma o punto delante.
in case (of)por si acaso
  • Precaución: Take an umbrella in case it rains.
  • ≠ if: no expresa condición sino prevención.

F. Cómo construir un mixto paso a paso
Esquema rápido para no confundir tiempos
1
Identifica el contraste temporal

Decide si la causa es pasada y la consecuencia presente (3 → 2), o al revés: característica/estado presente con consecuencia pasada (2 → 3).

causa pasada → efecto hoy rasgo presente → efecto pasado
2
Aplica el tiempo correcto a la if-clause

Si la causa es pasada → past perfect. Si es un rasgo o estado presente → past simple (o were).

If I had studied… If she were more cautious…
3
Aplica el tiempo correcto a la principal

Consecuencia presente → would + infinitive. Consecuencia pasada → would have + p.p.

I would be a doctor she wouldn't have crashed
4
Añade un marcador temporal explícito

Sin now / today / still / by now / last week el mixto puede confundirse con un Tipo 3 erróneo. El marcador es la señal que justifica el cambio de tiempo.

… now … today … by now … last week
5
Considera la inversión formal (C1)

Para Writing P2 sube el registro con Had + S + p.p. en lugar de If…had…: Had we acted then, we would be ahead now.

Had we acted, … Were she to apply, … Should you need…
📏 Test rápido: ¿puedes añadir now o last week sin que la frase chirríe? Si sí, el mixto funciona. Si no, vuelve a Tipo 2 o 3 puro.
